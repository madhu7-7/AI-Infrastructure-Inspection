# AI Infrastructure Inspection using YOLOv8

This project uses a YOLOv8 deep learning model to classify infrastructure images as **defective (positive)** or **non-defective (negative)**.

## Features
- Image classification using YOLOv8
- Custom dataset training
- Automated dataset organization
- Prediction script for testing new images

## Project Structure
datasets/ - training and testing images  
scripts/ - training script  
predict.py - run model predictions  
requirements.txt - required Python libraries  

## Installation

pip install -r requirements.txt

## Run Prediction

python predict.py

## Model
The model was trained using YOLOv8 classification architecture.
